Address Manager

ENTITY RX_DMA_CALYPTE_ADDR_MANAGER IS

This component manages addresses and pointer for each channel. Component is working as follows: After receiving address request for the specific channel, the HW pointers are increased and the address for storing the data in RAM is created. The addres is valid as soon as the corresponding area in a host memory is free.
