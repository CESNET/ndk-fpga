Channel Splitter

ENTITY TX_DMA_CHANNEL_SPLITTER IS

This component automatically splits incoming PCIe stream into one of the outputs to channels. The decision of a channel number is made using PCIe header fields like ADDRESS, BAR and BAR_APERTURE. The component also provides and indication on the output interfaces, if a current transaction contains DMA header (this transactions use different address range than ordinary data transactions).
