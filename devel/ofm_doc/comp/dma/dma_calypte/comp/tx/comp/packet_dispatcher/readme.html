Packet Dispatcher

ENTITY TX_DMA_PKT_DISPATCHER IS

This component dispatches the frames from data buffers accoring to available DMA Headers. Frames are dispatched from all channels in order in which DMA headers did come from the PCI Express. After dispatching a frame, the component issues an update of header and data pointers. If a channel is already stopped and DMA header for this channel occurs on the input of this component, this DMA header is dropped and no frame data are dispatched to the output as well as no update of pointers is issued.
