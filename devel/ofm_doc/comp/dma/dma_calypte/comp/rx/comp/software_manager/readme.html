Software Manager

ENTITY RX_DMA_SW_MANAGER IS

This entity initializes configuration and status registers for RX DMA Calypte controller. Its generic implementation allows to create configuration space for arbitrary number of channels. It also provides the access to registers from MI interface connected to designated software driver.
