Address Manager

ENTITY ADDR_MANAGER IS

This component manages ring buffers in RAM where the data are sent. Every channel has its own ring buffer.

Component is working as follows: After receiving request address for the specific channel, the HW pointers are increased and the address for storing the data in RAM is created. The design cna only manage constant size data.
